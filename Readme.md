# Link-me jQuery Plugin

**Link-me** is a simple jQuery plugin to attach links to elements

## Version
2.0.0

## NPM
[https://www.npmjs.com/package/link-me](https://www.npmjs.com/package/link-me)

## Installation
    npm install link-me

## Usage
    $('span').linkme({
			href: 'http://jyotsnasingh.com',
			text: 'Click Me',
			target : '_blank'
		});
